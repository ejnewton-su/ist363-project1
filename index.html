Project 1 -- OnsenUI

Onsen UI is an open-source UI framework and components for HTML5 hybrid mobile app development, based on PhoneGap / Cordova. It allows developers to create mobile apps using Web technologies like CSS, HTML5, and JavaScript.

OnsenUI is a HTML 5 & CSS framework created to make your life easier when it comes to web app creation!

Coding is the foundation of everything we do here at OnsenUI, making an adaptable and easy to use framework is key to our success.

We are optimized specifically for mobile applications, which makes your lives easier when transferring your web designs over to an easy to use mobile application.
